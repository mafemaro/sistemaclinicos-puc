# Sistema Clínico - PUC Minas

Projeto desenvolvido como parte da disciplina de Engenharia de Software I, com foco no subsistema de consultas e gestão de cadastros de pacientes.

## 🛠 Tecnologias Utilizadas
O projeto foi construído utilizando tecnologias nativas da web para garantir performance e clareza na implementação:

* **Frontend:** HTML, CSS.
* **Backend:** JavaScript.
* **Design:** Prototipação realizada no Figma para definição de fluxos e interface.

## 📋 Escopo do Projeto
Minha responsabilidade principal neste sistema compreende:
* **Fluxo de Cadastrar Paciente:** Implementação do formulário, validação de dados e a regra de negócio unívoca para CPF.
* **Dashboard de Consultas:** Criação da interface de visualização da agenda diária, profissionais e status de atendimento.
* **Dinâmicas de Sistema:** Implementação de feedbacks visuais (carregamento, sucesso e erro) para melhorar a experiência do usuário.

## 🚀 Como Executar
1. Clone este repositório em sua máquina local.
2. Certifique-se de que os arquivos `index.html`, `estilo.css` e `script.js` estejam na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador moderno para interagir com o protótipo funcional.

## ⏱ Estimativa de Prazo
O desenvolvimento desta etapa totalizou **30 horas de trabalho**, divididas em:

* **Prototipação e Design (10h):** Estruturação de telas e mapeamento da navegação.
* **Desenvolvimento (15h):** Codificação das interfaces e lógica de validação de CPF.
* **Testes e Validação (5h):** Ajustes de integração e testes de fluxo.
